LCD汉字计时报讯屏（使用C语言实现，IDE为keil5）

一、任务及设计要求
1）完成一个60分钟内计时器设计
2）在对应时间显示相应考试讯息在LCD点阵屏上
3）具备清屏、启动、暂停和关机等基本控制功能；
4）显示讯息能够实现左移、右移，上移，下移等跑马灯显示特性（根据显示方式的不同，可选择部分或全部功能实现）；
5）顺计时和60分钟倒计时切换，可实现声音提示和报警；
6）显示北京时间，可调整

二、设计实际所实现的功能
①显示北京时间并可调整；
②实现60min计时；
③可实现启动、暂停、关机功能并显示讯息能够显示跑马灯特性；
④能够顺计时和倒计时切换；
 
三、硬件设计及说明
4.1 学生设计所实际使用到的具体元器件清单
STM32F1开发板、LCD12864显示屏、KEY按键、定时器、蜂鸣器
4.2 硬件电路模块及功能说明
按键模块：
KEY0 和 KEY1 用作普通按键输入，分别连接在 PE4 和 PE3 上，没有使用外部上拉电阻，但是 STM32 的 IO 作为输入的时候，可以设置上下拉电阻，所以使用 STM32 的内部上拉电阻来为按键提供上拉。KEY_UP 按键连接到 PA0，它除了可以用作普通输入按键外，还可以用作 STM32 的唤醒输入。          
LED模块：
DS0为红色LED，DS1为绿色LED，电源指示灯为蓝色
蜂鸣器模块：
有源蜂鸣器是指自带了震荡电路的蜂鸣器，这种蜂鸣器一接上电就会自己震荡发声
LCD模块：
TFT_LCD是一个通用的液晶模块接口，图中的 T_MISO/T_MOSI/T_PEN/T_SCK/T_CS 连接在 MCU 的 PB2/PF9/PF10/PB1/PF11 上，这些信号用来实现对液晶屏的控制（支持电阻屏和电容屏），液晶复位信号 RESET 则是直接连接在开发板的复位按钮上，和MCU 共用一个复位电路
 
四、系统操作说明或使用说明
开发板上电后：可短按WK_UP在以下三种模式之间切换
模式一为时间模式，可短按KEY1改变选中位，短按或者长按KEY0改变选中位的值，长按WK_UP退出时钟设置，恢复正常计时；
模式二为计时60min：可短按KEY0启动计时器或者暂停计时器，长按KEY0重置计时器；短按KEY1切换提示语的循环方式（上下左右或者静止），长按KEY1切换顺逆计时；
模式三为设置开关机：长按WK_UP关机，再次长按开机，在关机状态下其他按键失效，时钟后台走时。
